# BillSplitter

Single-screen Android app that splits a restaurant bill among friends based on what each person ordered. Tax and tip distributed proportionally to each person's order total.

## Purpose

Demo showcase for JetBrains MCP plugins with Claude Code:

- **IDE Index MCP Server** — semantic code navigation
- **Debugger MCP Server** — breakpoints, stepping, variable inspection

Contains an intentional bug for live debugging demo.

## Tech Stack

- Kotlin
- Jetpack Compose (Material 3)
- Single-activity architecture

## Build

```bash
./gradlew :app:assembleDebug
```
